GrowTrack App

Panduan teknis dan operasional untuk aplikasi Java Desktop GrowTrack. Merangkum panduan antarmuka, tata cara input data medis balita, hingga arsitektur ERD Database.

1. Pengenalan & Hak Akses

Sistem GrowTrack mengimplementasikan *Role-Based Access Control* (RBAC) untuk membagi wewenang kerja di lingkungan Posyandu secara presisi. Otorisasi peran menentukan modul apa saja yang terbuka untuk menjaga kerahasiaan data balita.

Admin

Akses penuh sistem, manajemen database utama, dan pembuatan/pengaturan akun pengguna lain.

Super User

Bidan / Petugas

Aktor fungsional utama. Bertugas mengelola pendaftaran balita, pencatatan gizi, dan penjadwalan imunisasi.

Medis & Operasional

Kasir

Fokus eksklusif pada administrasi pendanaan kegiatan Pemberian Makanan Tambahan (PMT).

Keuangan PMT

2. Alur Kerja Utama

Penting untuk mengisi form secara berurutan guna menjaga relasi data (Foreign Key) di dalam database tetap valid.

1

Registrasi Orang Tua (Master)

Mendaftarkan identitas Ibu (Wajib) dan Ayah sebagai wali di form Orang Tua. Menghasilkan ID unik.

2

Pendaftaran Balita (Master)

Mendaftarkan nama anak, Tanggal Lahir, Jenis Kelamin, dan memilih Orang Tua anak tersebut dari database.

3

Pelayanan Gizi & Vaksin (Transaksi)

Bidan melakukan pengukuran (BB, TB, LK) di Form Pemeriksaan, atau menjadwalkan suntikan di Form Jadwal Imunisasi.

Pelaporan (Laporan)

Data terekam di Dashboard secara *real-time* dan siap diekspor menjadi dokumen PDF resmi bulanan.

3. Dashboard Utama

Indikator Metrik (Real-Time)

Data statistik pada Dashboard terhubung langsung dengan *Database Query*. Setiap Anda menekan tombol "Simpan" di form manapun, indikator ini akan memperbarui dirinya sendiri.

Tabel Peringatan Khusus

Dashboard menampilkan tabel *Quick View* yang secara otomatis menyaring dan memunculkan nama-nama balita yang memiliki status gizi Stunting, Gizi Kurang, atau Buruk dari pemeriksaan terakhir mereka agar menjadi prioritas pantauan Bidan.

4. Manajemen Master Data

Data Orang Tua

  • ID OtomatisSistem meng-generate (Misal: ORT-005).
  • Nama Ibu (Wajib)Harus diisi sebagai pemegang KIA utama. Nama Ayah opsional.
  • Validasi AngkaNo. Handphone akan menolak huruf untuk menjaga kebersihan database.

Data Balita

  • Relasi DropdownPilih Orang tua dengan mengetikkan nama Ibu/Ayah pada ComboBox.
  • Gender & Tanggal LahirPastikan data ini akurat 100%. Ini adalah nyawa dari algoritma kecerdasan buatan penentu Stunting nantinya.

5. Pemeriksaan Gizi Terpadu

Algoritma Smart Detector

Form Pemeriksaan adalah otak aplikasi. Bidan cukup mengetik Berat Badan dan menekan Enter ke Tinggi Badan.
Sistem Java akan langsung menarik usia anak (dalam hitungan bulan spesifik), membedah gender, dan menghitung Z-Score berdasarkan tabel referensi standar WHO Permenkes No.2/2020 tanpa menekan tombol hitung.

Z-Score = (Nilai Input - Median WHO) / Standar Deviasi

6. Metrik Lingkar Kepala (LK)

Mikrosefali (< -2 SD)

Kepala terlalu kecil. Indikasi perkembangan volume otak terhambat.

Normal (-2 SD s.d +2 SD)

Struktur tempurung otak tumbuh proporsional sesuai standar usianya.

Makrosefali (> +2 SD)

Kepala terlalu besar. Perlu deteksi dini risiko hidrosefalus.

7. Jadwal Vaksinasi

Pengingat cerdas bagi bidan agar jadwal imunisasi bayi tidak terlewat.

  • Master VaksinDaftarkan stok (BCG, DPT, Campak) di menu Master Data Vaksin.
  • Status "Belum"Jika di-set 'Belum', sistem akan menampilkan nama anak di Dashboard utama sebagai peringatan.

10. Instalasi Database & Arsitektur ERD

Panduan Konektivitas (XAMPP / MySQL)

  1. Pastikan servis Apache dan MySQL berjalan pada panel kendali XAMPP Anda.
  2. Akses `http://localhost/phpmyadmin` di browser.
  3. Buat database baru dengan nama pasti: db_posyandu.
  4. Lakukan Import file db_posyandu.sql ke dalam database tersebut.
  5. Jalankan Project melalui NetBeans IDE. Koneksi akan otomatis terbentuk via kelas KoneksiDB.java.
/src/konfigurasi/KoneksiDB.java
String url = "jdbc:mysql://localhost:3306/db_posyandu";
String user = "root";
String pass = "";

DriverManager.registerDriver(new com.mysql.cj.jdbc.Driver());
koneksi = DriverManager.getConnection(url, user, pass);
