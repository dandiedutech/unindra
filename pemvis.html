Dokumen Project Akhir (Java Netbeans)

SIP-Ruang & Sarpras

Platform Desktop berbasis Client-Server (Offline) untuk digitalisasi peminjaman fasilitas kelas dan sarana prasarana kampus.

100% Sesuai Syarat Project

Telah memenuhi syarat minimum Dosen: Terdapat Login, Menu, 6 Form (Master/Transaksi), dan 4 Report.

Glosarium & Istilah Teknis Project

  • TU: Tata Usaha (Admin Pengelola Server)
  • Mhs: Mahasiswa (Client / Pengguna Kiosk)
  • Kiosk: Aplikasi terminal mandiri untuk mahasiswa
  • CRUD: Create, Read, Update, Delete
  • ACC: Accepted (Pengajuan disetujui Admin)
  • Pending: Status menunggu keputusan/tinjauan
  • Qty: Quantity (Jumlah barang dipinjam)
  • PK (Primary Key): Kunci unik data di tabel
  • FK (Foreign Key): Kunci relasi antar tabel
  • JasperReports: Library Java untuk cetak PDF
